# E-commerce-Platform

This project is a simple e-commerce application built to practice backend development concepts such as product management, carts, and orders.
It focuses on core functionality rather than UI polish and is intended as a learning and portfolio project

## Features
- Product catalog

- Shopping cart

- Checkout and orders

- Basic inventory tracking

## Tech stack

- Language: Java 21

- Backend Framework: Spring Boot

- API: REST

- Database: PostgreSQL

- ORM: Spring Data JPA (Hibernate)

- Database Migrations: Flyway

- API Documentation: OpenAPI / Swagger (springdoc-openapi)

- Testing: JUnit 5

- Containerization: Docker (for local development and deployment)

## Optional / Planned (Future)

(to be added as the project evolves)
